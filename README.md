# Demo Dating Website ‚Äì Security Awareness Project

This project is a demo dating website developed for educational purposes to demonstrate common web security vulnerabilities. The application simulates how user details and uploaded photos can be captured during login and signup when proper security measures are not implemented. A ‚Äúhacked‚Äù message is displayed to highlight the potential risks of insecure authentication and data handling.

## Project Objective
The main goal of this project is to create awareness about web application security issues such as weak authentication, improper data validation, and insecure storage of user information. It is intended strictly for learning and demonstration purposes.

## Features
- User signup and login functionality  
- Captures user details and profile images  
- Demonstrates insecure data handling  
- Displays a hacked message to simulate a security breach  
- Simple UI for demonstration and testing

## Setup Instructions
1. Install XAMPP and start Apache and MySQL  
2. Copy the project folder into `htdocs`  
3. Create a MySQL database using phpMyAdmin  
4. Import or create required tables  
5. Configure database connection in PHP files  
6. Open the project in browser using `http://localhost:8080/project_name`
